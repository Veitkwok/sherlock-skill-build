# M05 · Physical / mechanical priority

- **id:** M05  
- **triggers:** EPS vs cash, quality of beat, “non-GAAP story”, undervalued claim  
- **when_not:** pure technical §12 with no fundamental claim  

## Method

Weight hard-to-manipulate mechanics over controllable narrative. Cash, inventory, receivables, real capex, ownership filings, and **IBKR-observed** tape/positioning beat guidance tone.

## US equities ops

Prefer L0 from:

| Physical | Source |
|----------|--------|
| CFO / FCF conversion | Web 10-Q/K |
| Inventory days, DSO | Web filings |
| Dilution / SBC | Web |
| Form 4 / 13F | Web |
| ADV, gap, volume climax | IBKR snapshot/history |
| IV/OI skew | IBKR options tools |

Deprioritize: adjective-heavy slides, ESG, unaudited “KPIs” without cash bridge.

## Falsifiers

- Cash conversion improves while narrative is bad (must raise grade weight on physical).  
- Physical series too short → DATA_GAP, not A on dim4.

## Anti-patterns

- “Adjusted EBITDA beat” as physical proof.  
- Fake options flow APIs — only IBKR-named fields.
