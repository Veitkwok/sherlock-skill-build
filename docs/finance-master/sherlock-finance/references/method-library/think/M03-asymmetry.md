# M03 · Asymmetry / symmetry-breaking

- **id:** M03  
- **triggers:** pattern break, peer divergence, “usually does X but now Y”  
- **when_not:** no historical baseline available (DATA_GAP — say so)  

## Method

Actively hunt **breaks** in symmetry: historical self-pattern, peer correlation, disclosure cadence, capital allocation consistency. Micro-breaks are high-value leads, not noise.

## US equities ops

Scan (Web + IBKR):

- Peer IBKR relative performance vs historical beta/co-move (from history packs)  
- Margin/FCF vs prior cycle same season  
- Guidance vocabulary change vs prior calls (transcript)  
- Insider pattern break (Form 4)  

Log each break as L0; do not “explain away” until Step 3.

## Falsifiers

- Break explained by sector-wide IBKR move shared by all peers.  
- One-off accounting reclass fully disclosed and cash-neutral.

## Anti-patterns

- Forcing symmetry (“must mean mean-reversion”).  
- Ignoring break because it hurts thesis (A2).
