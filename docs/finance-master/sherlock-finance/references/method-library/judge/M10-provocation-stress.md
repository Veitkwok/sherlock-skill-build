# M10 · Provocation / scenario stress

- **id:** M10  
- **triggers:** fragile guidance, “structural margin”, peak multiple, max undervalued claim  
- **when_not:** no load-bearing quantitative claim  

## Method

When passive L0 is insufficient, **force** weak joints with aggressive but explicit scenarios. Provocation is structured stress, not rhetoric.

## US equities ops

1. Identify load-bearing assumption (e.g. +200 bps margin, 30% growth forever).  
2. Build 2–3 adverse scenarios with numeric hits.  
3. Ask what filing/IBKR observation would show if scenario true.  
4. Search Web/IBKR for those traces; if absent, raise falsification dim or DATA_GAP.  
5. Record results in Step 3 notes; feed conf dim2.

Example stresses: price −15% on IBKR path, gross margin −150 bps, customer loss, funding freeze.

## Falsifiers

- Thesis survives quantified stress with physical L0 still intact.  

## Anti-patterns

- Vague “what if it crashes” without numbers.  
- Stress designed only to confirm bull case.
