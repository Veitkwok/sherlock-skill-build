# M01 · Observation before deduction

- **id:** M01  
- **triggers:** premature thesis, headline → view, “seems like”, skip filings  
- **when_not:** pure numeric quote already fully L0-logged under lite Step 0  

## Method

Perception layer (L0) is a hard firewall. No L1+ until observation is deliberately finished and logged. Exhaust relevant sources **within mode budget**, then deduce.

## US equities ops (IBKR / Web / X)

1. Complete Step 0 checklist for `mode_effective` (§1).  
2. L0 lines only: IBKR tape/levels; Web 10-Q/8-K/transcript snippets; optional X with URLs.  
3. Only then open hypotheses (Step 2).  

## Falsifiers

- Thesis stated before any sourced L0.  
- “Everyone knows” without citation.

## Anti-patterns

- Reading one press headline → directional call.  
- Treating X consensus as observation of fundamentals.

## Modern micro-example

ARR growth headline → first log revenue recognition note, deferred revenue vs backlog, SBC%, IBKR volume on print — **then** infer sustainability.
