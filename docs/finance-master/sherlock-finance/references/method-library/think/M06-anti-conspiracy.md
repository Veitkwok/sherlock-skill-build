# M06 · Anti-conspiracy baseline

- **id:** M06  
- **triggers:** fraud ring narrative, “auditors + banks + insiders all colluding”, market-wide cabal  
- **when_not:** simple single-firm incentive misalignment already explained  

## Method

Default: large multi-party secrets are unstable. Prefer mechanisms needing **limited coordination** — management incentives + accounting discretion + weak controls — unless strong multi-source L0 forces a wider plot.

## US equities ops

Rank explanations:

1. Incompetence / optimism bias  
2. Incentive-compatible earnings management (comp, covenants)  
3. Small-circle fraud (mgmt ± advisors)  
4. Wide collusion (require extraordinary independent chains)

Require more independent chains as conspiracy scope grows. A1 if scope ≫ evidence.

## Falsifiers

- Multiple independent whistleblowers + forensic cash proof + regulatory actions.  
- Then allow wider model under Occam cost.

## Anti-patterns

- Social media “everyone knows it’s a scam” as multi-party proof.  
- Skipping Form 4 / related-party L0.
