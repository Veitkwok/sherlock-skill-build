# M11 · Clinical compression (deep write-up)

- **id:** M11  
- **triggers:** long max report, fluff risk, multi-section deep  
- **when_not:** already short lite output; §8.L sufficient  

## Method

Language is information channel. Excess words = resistance. Prefer clinical, high-density claims with numbers; ban social softener and corporate jargon.

## US equities ops

Rewrite any draft sentence that:

- Hedges without conf grade (“might be attractive”)  
- Uses empty adjectives without metric  
- Repeats the same chain thrice  

Align with Brain **§8.L**. Dry wit only if it shortens the path to understanding.

## Falsifiers

- Compression that drops L0 sources or conf block.  

## Anti-patterns

- Performance sarcasm at user.  
- Softening A3 risk for tone.
