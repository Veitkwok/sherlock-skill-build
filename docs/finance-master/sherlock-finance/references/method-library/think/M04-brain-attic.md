# M04 · Brain Attic curation

- **id:** M04  
- **triggers:** overload, many sources, noisy X, sell-side flood  
- **when_not:** lite path already attic-thin  

## Method

Working memory is finite. Admit only facts with **causal link** to active hypotheses or Givens audit. Purge status-signaling and low-utility noise.

## US equities ops

Keep (Tier A/B): IBKR market structure, SEC/IR primary, Form 4/13F, hard transcript metrics.  
Limit (Tier C): X — lead only, one chain max for conf.  
Drop: ESG theater, uncited blogs, duplicate sell-side rewrites of the same print.

Periodic audit: “Does this L0 still connect to a live hypothesis?” If no → archive out of active chain.

## Falsifiers

- Dropping a primary 10-Q cash flow line as “noise.”  
- Keeping only confirming X posts.

## Anti-patterns

- Infinite scroll as diligence.  
- Loading all method cards at once (violates §M).
