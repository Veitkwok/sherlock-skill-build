# M08 · Rest Test / energy consistency

- **id:** M08  
- **triggers:** too-smooth story, relentless positivity, strategy that never costs effort  
- **when_not:** single dry filing fact pattern with no narrative load  

## Method

Ask: if this story were true, where would energy, capital, and attention **have to** go? If management narrative requires little painful tradeoff while financials show strain, consistency fails the Rest Test.

## US equities ops

Check consistency between:

- Verbal priorities (transcript)  
- Capex / R&D / buyback / debt paydown (cash flow statement — Web)  
- Comp metrics (proxy — Web)  
- IBKR: does the market price “effort” (volatility/volume) match claimed certainty?  

Flag “rests too easily” when hard capital allocation contradicts soft words.

## Falsifiers

- Capex and hiring (Web) match stated strategy; tape calm for good reason (low float event).  

## Anti-patterns

- Psychologizing CEOs without capital L0.  
- Rest Test as pure vibes (must tie to filings/IBKR).
