# M07 · Empty space as evidence

- **id:** M07  
- **triggers:** missing metric, no segment split, silence on customer, absent buyback update  
- **when_not:** company never reported that metric historically  

## Method

What **should** appear in the crime scene but does not is an active clue. Treat systematic absences as L0 “negative space,” then hypothesize why absence is rational for the true state.

## US equities ops

Examples of empty space:

- Growth company stops disclosing unit metrics  
- No 8-K where peers issue one on similar event  
- Guidance raises without cash bridge  
- “Strong demand” without backlog/RPO when previously given  
- IBKR: no volume confirmation on claimed breakout  

Procedure: list expected disclosures → mark present/absent → for each absence, 2 motives (benign vs adverse) → falsify.

## Falsifiers

- Absence explained by new accounting standard that removes the line for all peers.  
- Data exists in exhibit user/Web missed (search again before adverse claim).

## Anti-patterns

- Inventing what “must” be hidden without peer baseline.  
- Equating silence with guilt without incentive model.
